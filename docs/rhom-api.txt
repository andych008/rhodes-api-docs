# Rhom API

Reset the Rhodes model database. For examples, see [Resetting the Database in Using the Local Database with Rhom](http://docs.rhomobile.com/rhodes/rhom#resetting-the-database).

## database_fullclient_reset_and_logout

Equivalent to `Rhom::Rhom.database_full_reset(true)` followed by `SyncEngine.logout`.

## database_full_reset

Deletes all records from the property bag and model tables.

	:::ruby
	database_full_reset(reset_client_info, reset_local_models)

<table border="1">
<tr>
	<td><code>reset_client_info</code></td>
	<td>true to clean the client_info table, false otherwise.</td>
</tr>
<tr>
	<td><code>reset_local_models</code></td>
	<td>true to clean local (non-synced) models, false otherwise.</td>
</tr>
</table>

## database_full_reset_and_logout

Perform a full reset, then logout the RhoSync client.

## database_local_reset

Reset only local (non-sync-enabled) models.
