# Camera API

Use the camera on a device.

## get_camera_info

Get camera information. Returns max_resolution: a hash with width and height numeric values (returns NIL if the camera_type is not supported).

Syntax: get_camera_info(camera_type)

camera_type – main or front. Default = main. 

## choose_picture

Choose a picture from an album.

Syntax: choose_picture(camera_callback_url)

camera_callback_url - url to the callback method. Parameters in camera callback:

* status – “ok”, “cancel”, “error”  
* message – if status = error, contains message about error  
* image_uri – uri of saved image file  
* image_width – image width * “image_height” – image height  
* image_format – “png”, “jpg”

## take_picture

Take a picture.

Syntax: take_picture(camera_callback_url, options_hash)

camera_callback_url - url to the callback method. Parameters in camera callback:

* status – “ok”, “cancel”, “error”  
* message – if status = error, contains message about error 
* image_uri – uri of saved image file  
* image_width – image width * “image_height” – image height  
* image_format – “png”, “jpg”

options_hash - hash of options. Used only on iOS and Android.

 * camera_type – main or front. Default value = main.  
 * desired_width – desired width in pixels. Default value = max camera width.  
 * desired_height – desired height in pixels. Default value = max camera height.  
 * color_model - RGB or Grayscale. Default = RGB.  
 * format - image format. jpg or png. Default = jpg.  
 * enable_editing – boolean. Enables post photo capture image customizing; image will captured reduced to screen size (not full size). Supported only on iPhone. Default = true.
