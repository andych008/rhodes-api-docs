# Barcode API

Allows the device to recognize a barcode.

## barcode_recognize(image_file_full_path)

Recognizes a barcode on an image. Returns a string with recognized code, or emty string if the barcode is not recognized.

image_file_full_path - path to the image.

## take_barcode(url_for :action => :take_callback)

Android only. Opena a user interface for real-time barcode recognizing from the camera. If a barcode is found, user can confirm recognizing barcode or continue recognizing the barcode. If user confirmed or cancelled the callback will be called. 

callback: 
“status” – “ok” or “cancel”  
“barcode” – recognized barcode string
