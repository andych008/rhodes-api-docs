# AsyncHttp API - Connect Directly to Web Services

Asynchronously make calls to http services.

## AsyncHttp.get(url, headers, callback)

Perform an HTTP GET request to the specified URL.

url – URL of the request.  
headers – Hash of headers to send with the request.  
callback – Callback action to execute when the request is done.   
callback_param – (optional) Parameters to send to the callback. Parameters values should be url encoded with RhoSupport.url_encode.  
ZZZZZZZZ put in link to RhoSupport  
authentication – (optional) Send Basic Auth header with request.
:ssl_verify_peer – (optional) Verify SSL certificates, true by default.

The authentication parameter takes the form:

	:::ruby
	:authentication => {
	  :type => :basic, 
	  :username => "john", 
	  :password => "secret"
	}
