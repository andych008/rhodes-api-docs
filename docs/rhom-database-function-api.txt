# Rhom Database Functions

Reset and Seed the rhodes model database.

## database_fullclient_reset_and_logout

Equivalent to `Rhom::Rhom.database_full_reset(true)` followed by `SyncEngine.logout`.

## database_full_reset(reset_client_info, reset_local_models)

Deletes all records from the property bag and model tables.

reset_client_info - true if client_info table will be cleaned, false otherwise.
reset_local_models - true if local (non-synced models) will be cleaned, false otherwise.

## Rhom.database_full_reset_and_logout
